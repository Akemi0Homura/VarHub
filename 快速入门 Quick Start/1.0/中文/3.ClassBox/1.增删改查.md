ClassBox的增删改查和前面的Box在API层面几乎一样，因此这里仅仅快速带过：
```java
	//创建ClassBox对象
        Repository box= ClassBox.getClassBox();

        //添加整数10，第三个参数其实可以随便一个类
        //但是这个类最好是写当前存储这个数据的类，这样会更方便记忆并且符合逻辑
        //第四个参数是严格模式开关，true为开启，false为关闭，具体逻辑和Box基本一致
        box.addObject("a1",10,Boxtest.class,true);

	//创建ClassBox对象
    	Repository box= ClassBox.getClassBox();

	//这里打印来自Boxtest类的Integer中的a1数据，运行结果为10。
	//最后一个参数为严格模式开关
        System.out.println(box.getObject("a1",Integer.class,Boxtest.class,true));
```
最后再演示一下修改数据和删除数据：
```java
	//修改数据，数据的来源是Boxtest类的Integer中的a1数据，修改为30
        box.setObject("a1",30,Boxtest.class,true);

	//删除数据，删除数据的来源是Boxtest类的Integer中的a1数据
        box.removeObject("a1",Integer.class,Boxtest.class);
```
对象的操作和SimpleBox基本一致，这里不再演示
